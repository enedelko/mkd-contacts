# CLAUDE.md

Кворум-МКД — веб-приложение для сбора контактов собственников МКД, подготовки ОСС и расчёта кворума.

## Правила и ограничения

- **152-ФЗ:** ПДн (телефон, email, telegram_id, обращение) шифруются AES, поиск — по Blind Index (HMAC-SHA256). Мастер-ключ не в репозитории.
- **Без ФИО:** система не хранит ФИО; идентификация по кадастровому номеру помещения и контактным данным.
- **Аудит:** любое чтение или изменение ПДн пишется в audit_log.
- **Архитектура:** не вводить новые БД, очереди или протоколы — только PostgreSQL, Docker Compose, Nginx на хосте. См. docs/arch.md.
- **Миграции:** Alembic, выполняются автоматически при старте backend (entrypoint.sh). Новые миграции — через `alembic revision --autogenerate`.
- **API:** входные и выходные данные — JSON.

## Требования (SRS)

Перед реализацией фичи проверить спецификацию: docs/srs/00-INDEX.md (навигация по модулям SRS), docs/feature-list.md (статус фич: Готово / Бэклог / Отменено). Требования атомарны (SR-xxx).
